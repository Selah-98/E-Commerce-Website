# Khumalo Emporium

## Project Overview

Khumalo Emporium is a simple e-commerce web application developed as part of a school project.
The website was built using **Visual Studio 2022** and combines **HTML, CSS, and C# (ASP.NET)** to simulate an online platform where artisans can showcase and sell handmade products.

The goal of the project was to demonstrate the basics of web development, including front-end design, backend logic, and how a web application runs in a development environment.

---

## Technologies Used

* **Visual Studio 2022**
* **C# (ASP.NET)**
* **HTML**
* **CSS**
* **MySQL**

## How to Run the Project

Follow these steps to run the website on your computer:

### Step 1: Download or Clone the Project

Download the project files or clone the repository to your computer.

### Step 2: Open the Project in Visual Studio

1. Open **Visual Studio 2022**.
2. Click **Open a Project or Solution**.
3. Navigate to the project folder.
4. Select the **.sln (solution) file** and open it.

### Step 3: Build the Project

1. In Visual Studio, go to the top menu.
2. Click **Build**.
3. Select **Build Solution** to ensure all files compile correctly.

### Step 4: Run the Website

1. Click the **run (▶) button** at the top of Visual Studio.
2. Visual Studio will launch a browser window.
3. The website will open locally and you can interact with it.
